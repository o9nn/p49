# Pattern Dynamics AIML Chatbot Example

This example demonstrates how to use the Pattern Dynamics AIML files with Python.

## Prerequisites

```bash
pip install python-aiml
```

## Query Examples

Here are some example queries you can try:

### Meta Queries
- `WHAT IS PATTERN DYNAMICS`
- `HOW MANY PATTERNS ARE THERE`
- `LIST ALL PATTERNS`
- `WHAT ARE THE FIRST ORDER PATTERNS`
- `WHAT IS INTEGRAL SEMIOTIC REALISM`

### First-Order Patterns
- `WHAT IS RHYTHM`
- `WHAT IS POLARITY`
- `WHAT IS STRUCTURE`
- `WHAT IS EXCHANGE`
- `WHAT IS CREATIVITY`
- `WHAT IS DYNAMICS`

### Second-Order Patterns (42 patterns)
- Source Family: `ENERGY`, `PATTERN`, `POWER`, `TRANSFORMITY`, `RESOURCE`, `AUTOPOIESIS`
- Dynamics Family: `SYSTEM`, `SPONTANEITY`, `FEEDBACK`, `SYNERGY`, `AGENCY-COMMUNION`, `ITERATE`
- Creativity Family: `EVOLUTION`, `EMERGENCE`, `GROWTH`, `ADAPTATION`, `BIFURCATION`, `SEED`
- Exchange Family: `PROCESS`, `UNIQUENESS`, `TRADE`, `CAPTURE`, `BALANCE`, `CYCLE`
- Structure Family: `HOLARCHY`, `COMPLEXITY`, `NETWORK`, `HIERARCHY`, `HOLON`, `BOUNDARY`
- Polarity Family: `COMPETITION-COOPERATION`, `ORDER-CHAOS`, `FLOWS-STORES`, `INPUT-OUTPUT`, `CONCENTRATION-DIFFUSION`, `EXPAND-CONTRACT`
- Rhythm Family: `ENANTIODROMIA`, `SYNCHRONIZATION`, `PULSE`, `CADENCE`, `SWING`, `REPETITION`

### Query Types
For any pattern, try:
- `WHAT IS [pattern]`
- `TELL ME ABOUT [pattern]`
- `EXPLAIN [pattern]`
- `DESCRIBE [pattern]`
- `WHAT IS THE PRINCIPLE OF [pattern]`
- `[pattern] PRINCIPLE`
- `WHAT IS THE ROLE OF [pattern]` (second-order patterns)
- `WHAT IS THE DOMAIN OF [pattern]` (first-order patterns)
- `WHAT ORDER IS [pattern]`
- `WHAT FAMILY IS [pattern]` (second-order patterns)
