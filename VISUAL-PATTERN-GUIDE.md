When analyzing any system, look for:

- [ ] **Triadic structure** - Three distinct but related elements?
- [ ] **Infinity loop** - Continuous flow with crossing point?
- [ ] **Nested levels** - Fractally recursive organization?
- [ ] **Temporal pattern** - Rhythm + Repetition = Habit?
- [ ] **Evolution stages** - Movement through 0-6 cycles?
- [ ] **Active planes** - Which domains are engaged?
- [ ] **Viable system** - Operations/Coordination/Intelligence?
- [ ] **NonDual origin** - Source point from which all emerges?
- [ ] **Meta-morphosis** - Transformation between states?
- [ ] **Return path** - Integration back to source?
